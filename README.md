# Flutter App

Application mobile Flutter pour Android et iOS.

## Fonctionnalités
- Navigation entre plusieurs écrans
- Gestion basique d’état
- Affichage de contenus ou données (exemple)

## Technologies
- Flutter / Dart

## Structure
- `lib/main.dart` → point d’entrée
- `lib/screens/` → écrans
- `assets/` → images et icônes

## Installation
1. `git clone https://github.com/Ayaazouzi/flutter_app.git`
2. `cd flutter_app`
3. `flutter pub get`
4. `flutter run`
